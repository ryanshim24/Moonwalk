## Moonwalk - a multi-unit investment app

---

https://moonwalk.herokuapp.com/

Moonwalk is a single family unit calculator that provides the user to see if they get a good return of investment on a selected property. We built this app using Rails, ActiveRecord, Bcrypt, Nokogiri, AJAX, JSON, and JQuery 

#####Technology:
1. JavaScript
2. AJAX
3. Ruby
4. Rails
5. Nokogiri
6. JQuery
